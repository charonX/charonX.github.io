实例分析 – 预测精灵宝可梦进化后的CP值

目标

找到一个Function,输入一只pokemon然后输出这只pokemon进化后的CP值

  • 用X来表示一只进化前的pokemon
  • Xcp_{cp}​cp​​表示进化前的CP值
  • Xs_{s}​s​​ 表示物种
  • hp_{hp}​hp​​ 表示生命值
  • w_{w}​w​​ 表示重量
  • h_{h}​h​​ 表示高度
  • 进化后的CP值用y表示

下标表示完整的一个事物下的某一个部分

怎么做

三个步骤

  1. 找一个Model（a set of function)
  2. 从function set中找一个function出来判断它的好坏
  3. 从function set中找一个最好的function出来

Step 1： Model

y=b+wxcpy = b + wx_{cp}y=b+wx​cp​​ 代表function set

b,w 可以代入任何的值

这个model 可以称为一个Linear model

Linear model: y=b+∑wixiy=b+\sum w_{i}x_{i}y=b+∑w​i​​x​i​​

xix_{i}x​i​​: an attribute of input –> feature

wiw_{i}w​i​​: weight(权重)

b: bias (偏差)

Step 2: Goodness of Function

x1x^1x​1​​:上标表示完整的object编号, y^1\hat{y}^1​y​^​​​1​​：表示一个完整的Output Object,这里表示的是进化后的CP值

Training Data: 10 pokemons

定义一个Loss function L: input: a function, output: how bad it is

Step 3: Best Function 穷举所有的(w,b),带入这个Function中，output就会告诉你最好的(w,b)

Gradient Descent

Gradient Descent不止适用于解当前这个Function, 只要这个L是可微分的，Gradient Descent就可以帮助你解决

  1. 随机选取一个初始的点
  2. 在初始的这个点上对loss function微分，或者找它的切线斜率
  3. learning rate 是提前定好的数值
  4. 参数更新的幅度取决于微分的结果和learning rate的大小
  5. 重复执行直到找到global optimal
  6. Linear model不存在local optimal
